Dynamic Programming Table

Each cell dp[i][w] = max profit using the first i tasks with capacity w. Cells highlighted in green are on the backtracking path that recovers the optimal subset.

About the Algorithm

This solver uses bottom-up Dynamic Programming. The recurrence is: 

dp[i][w] = dp[i-1][w]                                    if r_i > w
dp[i][w] = max( dp[i-1][w],
                dp[i-1][w - r_i] + p_i )                  otherwise

Time complexity: O(n × W)  ·  Space complexity: O(n × W)  ·  Reduced space: O(W) with 1-D DP
